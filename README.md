# Flower Shop Telegram Bot 🌸

Telegram бот для цветочного магазина с интеграцией Google Sheets для управления инвентарем и использованием OpenAI для генерации ответов.

## Структура проекта

```
flower_shop_bot/
├── src/                        # Основной код проекта
│   ├── bot/                    # Модули Telegram бота
│   │   ├── handlers.py        # Обработчики сообщений
│   │   └── bot.py            # Основной класс бота
│   ├── services/              # Сервисные модули
│   │   ├── sheets_service.py # Работа с Google Sheets
│   │   ├── openai_service.py # Интеграция с OpenAI
│   │   └── channel_logger.py # Система логирования
│   ├── scripts/               # Вспомогательные скрипты
│   │   └── init_catalog.py   # Инициализация каталога
│   └── main.py               # Точка входа приложения
├── supabase/                  # Интеграция с Supabase
│   └── functions/            # Serverless функции
├── .env                      # Переменные окружения
└── requirements.txt          # Зависимости проекта
```

## Основные компоненты

### 1. Telegram Bot (`src/bot/`)
- `bot.py`: Основной класс бота, инициализация и конфигурация
- `handlers.py`: Обработчики сообщений и команд
  - Обработка текстовых сообщений
  - Поиск информации о цветах
  - Форматирование ответов

### 2. Сервисы (`src/services/`)

#### Google Sheets Service (`sheets_service.py`)
- Интеграция с Google Sheets API
- Методы для работы с инвентарем:
  - `get_inventory_item()`: Получение информации о товаре
  - `update_inventory_item()`: Обновление данных
  - `add_inventory_item()`: Добавление нового товара
- Структура таблицы:
  - Название товара
  - Количество
  - Цена
  - Описание
  - Категория

#### OpenAI Service (`openai_service.py`)
- Интеграция с OpenAI API
- Генерация ответов на основе контекста
- Системный промпт для точных ответов
- Обработка ошибок и повторные попытки

#### Channel Logger (`channel_logger.py`)
- Система логирования в Telegram канал
- Отслеживание ошибок и важных событий
- Мониторинг работы бота

### 3. Скрипты (`src/scripts/`)
- `init_catalog.py`: Инициализация структуры каталога в Google Sheets

## Конфигурация

### Переменные окружения (.env)
```
TELEGRAM_BOT_TOKEN=your_bot_token
TELEGRAM_CHANNEL_ID=your_channel_id
OPENAI_API_KEY=your_openai_key
GOOGLE_SHEETS_CREDENTIALS_FILE=path_to_credentials
GOOGLE_SHEETS_SPREADSHEET_ID=your_spreadsheet_id
```

### Google Sheets
Структура таблицы:
- Sheet: "Inventory"
- Колонки:
  - Item Name
  - Quantity
  - Price
  - Description
  - Category

## Основные функции

### Обработка запросов
1. Получение сообщения от пользователя
2. Анализ текста на наличие ключевых слов
3. Поиск информации в Google Sheets
4. Генерация ответа через OpenAI
5. Отправка форматированного ответа

### Работа с инвентарем
1. Автоматическое обновление при изменениях
2. Проверка наличия товаров
3. Обновление цен и описаний
4. Логирование изменений

## Запуск проекта

1. Установка зависимостей:
```bash
pip install -r requirements.txt
```

2. Настройка переменных окружения:
- Создать `.env` файл
- Заполнить все необходимые переменные

3. Инициализация каталога:
```bash
python src/scripts/init_catalog.py
```

4. Запуск бота:
```bash
python src/main.py
```

## Логирование

Все важные события логируются в специальный Telegram канал:
- Ошибки API
- Изменения в инвентаре
- Проблемы с подключением
- Важные взаимодействия с пользователями
